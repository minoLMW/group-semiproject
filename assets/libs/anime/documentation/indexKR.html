오프셋은 마지막 에니메이션에 상대적이거나 전체 타임라인에 절대적일 수 있습니다.

Callbacks & Promises

Update

애니메이션 재생이 시작되자마자 모든 프레임에서 callback이 트리거 됩니다.

Type Parameters Info
Function animation 현재 애니메이션 개체 반환

Begin & Complete

begin() 애니메이션 재생이 시작되면 콜백이 한 번 트리거 됩니다.

complete() 애니메이션이 완료되면 콜백이 한 번 트리거 됩니다.

Both begin() and complete() callbacks are called if the animation's duration is 0.

Type Parameters Info
Function animation Return the current animation Object

loopBegin & loopComplete

loopBegin() callback은 루프가 '시작'될 때마다 한 번씩 트리거됩니다.

loopComplete() callback은 루프가 '완료'될 때마다 한 번씩 트리거됩니다.

Type Parameters Info
Function animation 현재 애니메이션 개체 반환

Change

애니메이션의 지연과 종료 지연 사이의 모든 프레임에서 callback이 트리거됩니다. DELAY 와 END DELAY.

Type Parameters Info
Function animation 현재 에니메이션 개체 반환

changeBegin & changeComplete

changeBegin() 애니메이션이 변경되기 시작할 때마다 callback이 트리거됩니다.

changeComplete() 애니메이션 변경이 중지될 때마다 callback이 트기거됩니다.

Animation direction will affect the order in which changeBegin() and changeComplete() are triggerd.

Type Parameters Info
Function animation Return the current animation Object

Finished promise

모든 애니메이션 인스턴스는 애니메이션이 완료되면 다음 선언한 이벤트를 실행시킵니다. 

animation.finished.then(function() {
  // Do things...
});

path 애니메이션의 대한 자세한 정보는 https://css-tricks.com/svg-line-animation-works/ 에서 확인하세요.

Easings

Linear

애니메이션에 Easing을 적용하지 않습니다.
불투명도 및 색상 전환에 모두 사용합니다. 

easing: 'linear'

Penner's functions

Robert Penner's Easing 기능 내장. 자세한 내용은 http://robertpenner.com/easing/ 참고.

Easings 관하여 https://easings.net 에서 확인하세요. 

easing: 'easeInOutSine'

사용가능 Easings :

in out in-out out-in
'easeInQuad' 'easeOutQuad' 'easeInOutQuad' 'easeOutInQuad'
'easeInCubic' 'easeOutCubic' 'easeInOutCubic' 'easeOutInCubic'
'easeInQuart' 'easeOutQuart' 'easeInOutQuart' 'easeOutInQuart'
'easeInQuint' 'easeOutQuint' 'easeInOutQuint' 'easeOutInQuint'
'easeInSine' 'easeOutSine' 'easeInOutSine' 'easeOutInSine'
'easeInExpo' 'easeOutExpo' 'easeInOutExpo' 'easeOutInExpo'
'easeInCirc' 'easeOutCirc' 'easeInOutCirc' 'easeOutInCirc'
'easeInBack' 'easeOutBack' 'easeInOutBack' 'easeOutInBack'
'easeInBounce' 'easeOutBounce' 'easeInOutBounce' 'easeOutInBounce'

Cubic Bézier Curve

사용자 정의 베지어 곡선 cubicBezier(x1, y1, x2, y2).
(커스텀 가능한 Easing 모션) 

easing: 'cubicBezier(.5, .05, .1, .3)'

You can use Bézier curves generator like Ceaser to generate your curves coordinates.

Spring

Spring 물리학을 기반으로 한 Easing 

easing: 'spring(mass, stiffness, damping, velocity)'

mass = 질량(움직이는 활동량이 무거워짐),
stiffness = 강성(스타트 지점 가속),
damping = 감쇠(끝 지점 감속),
velocity = 속도

Spring 애니메이션의 지속 시간은 Spring 파라미터에 의해 정의됩니다.
애니메이션 duration 매개 변수는 고려되지 않습니다.

Parameter Default Min Max
Mass 1 0 100
Stiffness 100 0 100
Damping 10 0 100
Velocity 0 0 100

Elastic

탄성 애니메이션이 붙은 easing.

easing: 'easeOutElastic(amplitude, period)'
in out in-out out-in
'easeInElastic' 'easeOutElastic' 'easeInOutElastic' 'easeOutInElastic'
Parameter Default Min Max Info
Amplitude 1 1 10 원곡선의 오버슈트를 제어합니다. 이 숫자가 클 수록 오버슈트가 더 많습니다.
Period .5 0.1 2 원곡선이 앞뒤로 이동하는 횟수를 제어합니다. 이 숫자가 작을수록 왔다 갔다 하는 횟수가 많아진다

Steps

애니메이션이 종료 값에 도달하는데 걸리는 점프 횟수를 정의합니다. 

easing: 'steps(numberOfSteps)'
Parameter Default Min Max
Number of steps 10 1 ∞

Custom easing function

사용자 지정 Easing은 FUNCTION BASED VALUE 섹션을 참고하세요. 

easing: function() { return function(time) { return time * i} }
Parameter Info
Time Return the current time of the animation

Helpers

remove

실행 중인 애니메이션 또는 시간 표시 막대에서 대상을 제거합니다.
대상 매개 변수는 TARGETS 속성과 동일한 값을 사용합니다.

모든 활성 애니메이션에서 대상을 제거합니다.

anime.remove(targets);

단일 애니메이션 또는 타임라인에서 대상을 제거합니다.

animation.remove(targets);

get

요소의 원래 값을 변환합니다.

anime.get(target, propertyName, unit);

anime.js는 https://developer.mozilla.org/fr/docs/Web/API/Window/getComputedStyle 문서를 사용하여 원본 CSS에 접근하기 때문에. 값은 거의 항상 'px'로 변환되며, 세 번째(선택사항) 인수는 원하는 단위로 값을 변환합니다. 

anime.get(domNode, 'width', 'rem');
Parameter Type Info
target 'string', var 모두 유효한 TARGETS를 사용할 수 있습니다.

CSS 변환 : 인라인 값만 엑세스할 수 있습니다.

set

지정된 대상으로 값을 즉시 설정합니다. 

anime.set(targets, {property: value});
Parameters Types Info
target(s) 'string', var 모두 유효한 TARGETS를 사용할 수 있습니다.
values object 모두 유효한 properties를 사용할 수 있습니다.

random

특정 범위 내의 임의의 정수를 반환합니다. 

anime.random(minValue, maxValue);

tick

외부 요청 requestAnimationFrame를 사용하여 애니메이션을 재생합니다. 

animation.tick(time)

anime.js 내장 RAF 루프가 시작되지 않도록 autoplay 매개 변수를 false로 설정하는 것을 잊지 마십시오.

running

현재 실행 중인 모든 활성 anime.js 인스턴스의 Array를 반환 합니다. 

anime.running

    

  

  





  
Suspend on visibility change

  

    
anime.suspendWhenDocumentHidden = false; // default true

    

      기본적으로 모든 애니메이션은 탭을 전환할 때 일시 중지됩니다. 사용자가 모든 것을 보고 애니메이션의 중요한 부분을 놓치지 않으려는 경우 유용합니다.

      그러나 백그라운드에서 지속적으로 재생할 수 있는 비디오 또는 오디오 트랙과 같이 애니메이션이 중단 없이 정상적으로 실행되도록 선택할 수 있습니다.
